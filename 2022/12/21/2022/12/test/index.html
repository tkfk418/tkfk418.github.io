Git & Github 설정

다운로드

  • 다운로드 되고 난 후 변경할 것 없이 설치하면 됨
  • 설치 후 바탕화면에서 마우스 우 클릭하여 git bash here 클릭
  • git version이 아래와 같이 나오면 설치 성공
